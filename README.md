# bird

Projet React + Vite minimal pour démo interactive « bird ».

Pour démarrer:

```bash
npm install
npm run dev
```

Navigation: swipe gauche/droite ou flèches clavier.

Fichiers importants:
- `src/App.jsx` : swipe navigation
- `src/components/Scene.jsx` : affiche vidéo, texte, questions, charge mini-jeu
- `src/components/MiniGames/*` : mini-jeux simples
